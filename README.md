# nuxt-vue

> Vue+Vuex服务端渲染模板
## 模块

* Sass
* nuxt@1.4.0

## 安装运行

``` bash
# 安装依赖
$ npm install # Or yarn install


# serve with hot reload at localhost:6061
$ npm run dev

# 产品发布环境
$ npm run build
$ npm start

# 构建静态页面
$ npm run generate
```
## 修改端口号

> 修改package.json文件

``` bash
"config": {
    "nuxt": {
      "host": "0.0.0.0",
      "port": "6061"  #端口号
    }
},
```
